Template Installation

  1. FTP Upload:
    • Open up your FTP manager and connect to your hosting
    • Browse to required directory (Normally public_html)
    • Upload the files inside aram folder.

Pages

    • index.html is for home page
    • about.html is for about us page
    • service.html is for service page
    • pricing.html is for pricing page
    • gallery.html is for gallery page
    • blog.html is for blog page
    • single.html is for single blog page
    • contact.html is for contact page
